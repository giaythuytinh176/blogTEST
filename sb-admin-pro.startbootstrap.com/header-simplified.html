Simplified Header

A simplified page header for use with the dashboard layout
Simplified Page Header Option
The simplified page header options allows you to remove the negative offset of the default page header. Just remove the negative margin from the .page-header element and add a top margin to the initial container within the main element. This will result in a simplified page header, that still includes the option for an icon and breadcrumbs.
